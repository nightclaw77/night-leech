# 📋 چک‌لیست API Keys - Night Leech Bot

## ⚠️ از سمت شما چه کارهایی باید انجام شود؟

برای راه‌اندازی ربات، به اطلاعات زیر نیاز داریم. لطفاً این موارد را تهیه کنید:

---

## 1️⃣ BOT TOKEN (الزامی) ⭐

**مرحله:**
1. تلگرام رو باز کنید
2. به @BotFather بروید
3. دستور `/newbot` را ارسال کنید
4. نام ربات را وارد کنید (مثلاً: Night Leech)
5. یوزرنیم ربات را وارد کنید (باید به bot ختم شود، مثل: night_leech_bot)
6. توکن دریافتی را کپی کنید

**فرمت توکن:**
```
123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11
```

**🔴 مهم:** این توکن را **فقط به من** بدهید، در جایی عمومی نگذارید!

---

## 2️⃣ TELEGRAM API Credentials (الزامی) ⭐

**مرحله:**
1. به https://my.telegram.org بروید
2. با شماره تلگرام خود لاگین شوید
3. روی "API Development tools" کلیک کنید
4. یک اپلیکیشن جدید بسازید:
   - نام: Night Leech Bot
   - توضیحات: Telegram Mirror/Leech Bot
5. `api_id` و `api_hash` را کپی کنید

**🔴 مهم:** 
- شماره تلگرام باید روی گوشی فعال باشد
- در صورت نیاز به OTP، شماره تماس نیاز است

---

## 3️⃣ USER ID (الزامی) ⭐

**مرحله:**
1. به @userinfobot بروید
2. دستور `/start` را ارسال کنید
3. آی‌دی عددی خود را کپی کنید (مثلاً: `971043547`)

## 📞 راهنما

اگر در هر مرحله مشکل دارید:

1. **BotFather کار نمی‌کنه:**
   - برنامه تلگرام را آپدیت کنید
   - از نسخه جدید تلگرام استفاده کنید

2. **my.telegram.org باز نمی‌شه:**
   - VPN روشن کنید
   - کوکی‌ها را پاک کنید

3. **Userinfobot جواب نمی‌ده:**
   - ربات را استارت کنید
   - چند دقیقه صبر کنید و دوباره امتحان کنید
