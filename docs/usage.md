# 📖 راهنمای استفاده - Night Leech Bot

## دستورات ربات

### دانلود و آپلود به تلگرام

| دستور | توضیح |
|-------|-------|
| `/leech <link>` | دانلود لینک و آپلود به تلگرام |
| `/mirror <link>` | دانلود و آپلود (بسته به تنظیمات) |
| `/tgleech <link>` | دانلود مستقیم به تلگرام |

### مدیریت

| دستور | توضیح |
|-------|-------|
| `/status` | نمایش وضعیت دانلودهای فعال |
| `/cancel <id>` | لغو دانلود با ID |
| `/cancelall` | لغو همه دانلودها |
| `/settings` | تنظیمات کاربر |
| `/help` | راهنما |

### مدیر (Owner)

| دستور | توضیح |
|-------|-------|
| `/restart` | ریستارت ربات |
| `/backup` | بکاپ تنظیمات |
| `/stats` | آمار استفاده |

---

## انواع لینک‌های پشتیبانی شده

- ✅ **تورنت**: لینک `.torrent` یا لینک مغناطیسی
- ✅ **لینک مستقیم**: HTTP/HTTPS از سایت‌های مختلف
- ✅ **Mega.nz**: لینک‌های Mega
- ✅ **YouTube**: ویدیوهای یوتیوب
- ✅ **تلگرام**: فایل‌های تلگرام

---

## مثال‌های استفاده

```
/leech https://example.com/file.mkv

/leech magnet:?xt=urn:btih:...

/mirror https://mega.nz/folder/...

/ytdl https://youtube.com/watch?v=...
```

## محدودیت‌ها

| نوع | محدودیت |
|-----|---------|
| حجم فایل | 2 GB (4 GB با Premium) |
| همزمانی | 3 دانلود همزمان |
| آپلود روزانه | نامحدود |

---

## نکات

1. فایل‌ها بعد از آپلود از سرور پاک می‌شوند
2. می‌توانید فایل‌ها را از Saved Messages دانلود کنید
3. برای فایل‌های بزرگ‌تر از 2GB، از Google Drive استفاده کنید
